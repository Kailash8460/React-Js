- TAILWIND CSS INSTALLATION TO ANY PROJECT (Tailwind CSS in Vite App)

1. Create your project
Start by creating a new Vite project if you donâ€™t have one set up already.
npm create vite 
(select React Js which we are currently working on it)

2. Install Tailwind CSS
Install tailwindcss and @tailwindcss/vite via npm.
npm install tailwindcss @tailwindcss/vite

3. Configure the Vite plugin
Add the @tailwindcss/vite plugin to your Vite configuration. 
(in vite.config.ts file)
import { defineConfig } from 'vite'
import tailwindcss from '@tailwindcss/vite' // this one and...

export default defineConfig({
  plugins: [
    tailwindcss(), // this 2 lines need to add there
  ],
})

4. Import Tailwind CSS - Utility and Mobile first CSS
Add an @import to your CSS file that imports Tailwind CSS. 
(you can add this directly in index.css file so that you can access it anywhere)
@import "tailwindcss";
